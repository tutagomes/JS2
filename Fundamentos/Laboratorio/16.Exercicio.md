# Exercício 



Utilizando os sites de geração de objetos (https://mockaroo.com/) e também o servidor https://www.npmjs.com/package/json-server, vamos criar uma interface web para realizar o CRUD no pequeno banco, utilizando Axios, tipos e também um formulário para recolher as informações.

Vamos criar então uma lista de tarefas, com opções de adicionar tarefa por título, incluindo também data de criação, data de entrega e se está concluída ou não.

Um formulário deve ser exibido na página para que o usuário consiga entrar com os dados, e ao apertar um botão, as informações devem ser enviadas para o JSON-Server e posteriormente mostradas em uma lista na tela.

Na lista, deve ser possível alterar o estado - feita ou não - além de poder excluir a tarefa.
